# 🎓 We Can 2 English Class - AI-Powered Learning Platform

تطبيق تعليمي ذكي لتعلم اللغة الإنجليزية مدعوم بالذكاء الاصطناعي

## 🌟 المميزات

✅ **تصميم عصري وجذاب** - واجهة مستخدم جميلة وسهلة الاستخدام  
✅ **ثنائي اللغة** - دعم كامل للعربية والإنجليزية  
✅ **متجاوب تماماً** - يعمل بكفاءة على جميع الأجهزة والشاشات  
✅ **تفاعلي وديناميكي** - أزرار وحقول إدخال حية  
✅ **مدعوم بالذكاء الاصطناعي** - تكامل مع ChatGPT و Google Gemini  
✅ **مواضيع متنوعة** - 10 مواضيع تعليمية مختلفة

## 📚 المواضيع المتاحة

1. 🎭 Feeling (المشاعر والعواطف)
2. 👕 Things we wear (الملابس والأزياء)
3. 🏃 Things we do (الأفعال والأنشطة)
4. 🌳 Beautiful nature (الطبيعة والبيئة)
5. 👥 Friends, actions, things (الأصدقاء والتفاعلات)
6. 🦁 Wild animals (الحيوانات البرية)
7. 🏠 My house (البيت والأثاث)
8. 📦 My things (الأشياء الشخصية)
9. 🌅 My day, animals, places (اليوم والحيوانات والأماكن)
10. 🚗 From here to there (التنقل والاتجاهات)

## 🛠️ التكنولوجيا المستخدمة

- **HTML5** - للهيكل والمحتوى
- **CSS3** - للتصميم والرسوميات
- **JavaScript** - للوظائف التفاعلية
- **AI Integration** - ChatGPT و Google Gemini

## 📁 الملفات الرئيسية

- `index.html` - الصفحة الرئيسية
- `style.css` - أنماط التصميم
- `script.js` - الوظائف الديناميكية

## 🚀 كيفية الاستخدام

1. أدخل اسمك في حقل "Student Name"
2. اختر الموضوع الذي تريد تعلمه
3. اضغط زر "توليد البرومبت"
4. يمكنك فتح ChatGPT أو Gemini للحصول على درس تفاعلي

## 💡 الميزات المتقدمة

- **Smart Prompts**: تلقائياً يتم إنشاء prompts مخصصة بناءً على الموضوع واسم الطالب
- **AI Integration**: ربط مباشر مع أدوات الذكاء الاصطناعي الشهيرة
- **Responsive Design**: تصميم يتكيف مع جميع أحجام الشاشات
